# gurugram-house-price-analysis

# 🏠 Gurugram House Price Analysis & Machine Learning

## 📊 Project Overview

This project focuses on analyzing housing prices in Gurugram and applying data preprocessing and machine learning techniques to understand key factors influencing property prices.

It demonstrates a complete data science workflow — from data cleaning and exploration to feature engineering and model building.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📁 Dataset

* File: `housing.csv`
* Contains:

  * Area (sq ft)
  * Location
  * Price
  * Property features

---

## 🔄 Workflow

### 🔹 Data Analysis (EDA)

* Understanding dataset structure
* Identifying missing values
* Exploratory Data Analysis
* Data visualization

### 🔹 Data Preprocessing

* Handling missing values
* Feature scaling
* Encoding categorical variables

### 🔹 Feature Engineering

* Creating new meaningful features
* Transforming variables

### 🔹 Machine Learning

* Train-test split
* Basic model implementation
* Pipeline creation using Scikit-learn
